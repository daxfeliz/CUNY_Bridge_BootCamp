General Information

The main purpose of the group is to:

  • Show tools and applications that are useful for research
  • Learn how to apply such tools to projects, especially using Python
  • Teach good coding practices
  • Create a space where one can ask questions and easily get help from students, post-docs, and faculty
  • Improve your coding skills, and have fun while doing it!

The Github repository for this class is: https://github.com/daxfeliz/cunybridgebootcamp
